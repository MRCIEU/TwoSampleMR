# I-squared calculation

This function calculates the \\I^2\\ statistic. To use it for the
\\I^2\_{GX}\\ metric ensure that the effects are all the same sign (e.g.
`abs(y)`).

## Usage

``` r
Isq(y, s)
```

## Arguments

- y:

  Vector of effects.

- s:

  Vector of standard errors.

## Value

Isq value

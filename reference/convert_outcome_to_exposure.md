# Convert outcome data to exposure data

Helper function to convert results from
[`extract_outcome_data()`](https://mrcieu.github.io/TwoSampleMR/reference/extract_outcome_data.md)
to `exposure_dat` format.

## Usage

``` r
convert_outcome_to_exposure(outcome_dat)
```

## Arguments

- outcome_dat:

  Output from
  [`extract_outcome_data()`](https://mrcieu.github.io/TwoSampleMR/reference/extract_outcome_data.md).

## Value

data frame

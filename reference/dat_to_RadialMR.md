# Convert dat to RadialMR format

Creates a list of RadialMR format datasets for each exposure-outcome
pair.

## Usage

``` r
dat_to_RadialMR(dat)
```

## Arguments

- dat:

  Output from
  [`harmonise_data()`](https://mrcieu.github.io/TwoSampleMR/reference/harmonise_data.md).

## Value

List of RadialMR format datasets

# Perform MR Steiger test of directionality

A statistical test for whether the assumption that exposure causes
outcome is valid.

## Usage

``` r
directionality_test(dat)
```

## Arguments

- dat:

  Harmonised exposure and outcome data. Output from
  [`harmonise_data()`](https://mrcieu.github.io/TwoSampleMR/reference/harmonise_data.md).

## Value

List

# Generate odds ratios

This function takes b and se from
[`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md) and
generates odds ratios and 95 percent confidence intervals.

## Usage

``` r
generate_odds_ratios(mr_res)
```

## Arguments

- mr_res:

  Results from
  [`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md).

## Value

data frame

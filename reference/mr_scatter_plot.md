# Create scatter plot with fitted lines showing the causal effect estimate for different MR estimators

Create scatter plot with fitted lines showing the causal effect estimate
for different MR estimators.

## Usage

``` r
mr_scatter_plot(mr_results, dat)
```

## Arguments

- mr_results:

  Output from
  [`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md).

- dat:

  Output from
  [`harmonise_data()`](https://mrcieu.github.io/TwoSampleMR/reference/harmonise_data.md).

## Value

List of plots

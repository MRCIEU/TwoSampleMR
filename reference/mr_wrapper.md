# Perform full set of MR analyses

Perform full set of MR analyses

## Usage

``` r
mr_wrapper(dat, parameters = default_parameters())
```

## Arguments

- dat:

  Output from
  [`harmonise_data()`](https://mrcieu.github.io/TwoSampleMR/reference/harmonise_data.md).

- parameters:

  Parameters to pass to MR functions. Output from
  [`default_parameters()`](https://mrcieu.github.io/TwoSampleMR/reference/default_parameters.md)
  used as default.

## Value

list

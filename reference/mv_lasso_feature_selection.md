# Apply LASSO feature selection to mvdat object

Apply LASSO feature selection to mvdat object

## Usage

``` r
mv_lasso_feature_selection(mvdat)
```

## Arguments

- mvdat:

  Output from
  [`mv_harmonise_data()`](https://mrcieu.github.io/TwoSampleMR/reference/mv_harmonise_data.md).

## Value

data frame of retained features

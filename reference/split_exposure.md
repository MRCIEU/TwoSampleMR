# Split exposure column

This function takes the exposure column from the results generated by
[`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md) and
splits it into separate columns for 'exposure name' and 'id'.

## Usage

``` r
split_exposure(mr_res)
```

## Arguments

- mr_res:

  Results from
  [`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md).

## Value

data frame

# Split outcome column

This function takes the outcome column from the results generated by
[`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md) and
splits it into separate columns for 'outcome name' and 'id'.

## Usage

``` r
split_outcome(mr_res)
```

## Arguments

- mr_res:

  Results from
  [`mr()`](https://mrcieu.github.io/TwoSampleMR/reference/mr.md).

## Value

data frame

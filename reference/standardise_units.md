# Try to standardise continuous traits to be in standard deviation units

Uses
[`estimate_trait_sd()`](https://mrcieu.github.io/TwoSampleMR/reference/estimate_trait_sd.md).

## Usage

``` r
standardise_units(dat)
```

## Arguments

- dat:

  Output from
  [`harmonise_data()`](https://mrcieu.github.io/TwoSampleMR/reference/harmonise_data.md).

## Value

Data frame
